Never use em-dashes (—) or double hyphens (--) in any output.

**Why:** User considers them uncivilized punctuation. Use proper sentences, commas, semicolons, colons, or parentheses instead.

**How to apply:** Anywhere you would use an em-dash or double hyphen, restructure the sentence. Use a period and start a new sentence, use parentheses for asides, use a colon to introduce, or use a comma for light pauses.
